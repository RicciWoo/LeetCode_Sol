## 96. Unique Binary Search Trees
- 96. Unique Binary Search Trees [Dynamic Programming] [Tree] [Medium]

#### Tags
- [Dynamic Programming] [Tree] [Medium]

#### Problem
Given n, how many structurally unique BST's (binary search trees) that store values 1...n?  
For example,  
Given n = 3, there are a total of 5 unique BST's.
>   1         3     3      2      1  
>    \       /     /      / \      \  
>     3     2     1      1   3      2  
>    /     /       \                 \  
>   2     1         2                 3

#### Data Structure
- Tree

#### Algorithm/Method
- Dynamic Programming

#### Input
- `int n`

#### Output
- `int`

#### Solution
``` C++
class Solution {
public:
    int numTrees(int n) {
        if (n <= 0) return 0;
        if (n == 1) return 1;
        vector<int> f(n + 1, 0);
        f[0] = 1;
        f[1] = 1;
        for (int i = 2; i <= n; i++)
            for (int j = 1; j <= i; j++)
                f[i] += f[j - 1] * f[i - j];
        return f[n];
    }
};
```

#### Boudary Checking
- `if (n <= 0) return 0;`
- `if (n == 1) return 1;`

#### Time Complexity
- $O(n^2)$

#### Space Complexity
- $O(n)$
