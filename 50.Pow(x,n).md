## 50. Pow(x, n)
- 50. Pow(x, n) [Math] [Binary Search] [Medium]

#### Tags

#### Problem
Implement pow(x, n).

Example 1:  
Input: 2.00000, 10  
Output: 1024.00000

Example 2:  
Input: 2.10000, 3  
Output: 9.26100

#### Data Structure
- 

#### Algorithm/Method
- Binary Search

#### Input
- `double x, int n`

#### Output
- `double`

#### Solution
``` C++
class Solution {
public:
    double myPow(double x, int n) {
        if (n < 0) return 1.0 / power(x, -n);
        else return power(x, n);
    }
    
private:
    double power(double x, int n) {
        if (n == 0) return 1.0;
        double v = power(x, n / 2);
        if (n % 2 == 0) return v * v;
        else return v * v * x;
    }
};
```
