## 50. Pow(x, n)
- 50. Pow(x, n) [Math] [Binary Search] [Medium]

#### Tags
- [Math] [Binary Search] [Medium]

#### Problem
Implement pow(x, n), which calculates x raised to the power n (xn).

Example 1:

    Input: 2.00000, 10
    Output: 1024.00000

Example 2:

    Input: 2.10000, 3
    Output: 9.26100

Example 3:

    Input: 2.00000, -2
    Output: 0.25000
    Explanation: $2^{-2} = 1/2^2 = 1/4 = 0.25$

**Note**:

- -100.0 < x < 100.0
- n is a 32-bit signed integer, within the range $[−2^{31}, 2^{31} − 1]$

#### Solution
``` C++
class Solution {
public:
    double myPow(double x, int n) {
        long long m = n;
        if (m < 0) {
            return 1. / _pow(x, -m);
        }
        return _pow(x, m);
    }

private:
    double _pow(double x, long long m) {
        if (m == 0) {
            return 1.;
        }
        
        double v = _pow(x, m / 2);
        if (m % 2 == 0) {
            return v * v;
        }
        return v * v * x;
    }
};
```

#### Time Complexity
- O(log n)

#### Space Complexity
- O(1)
